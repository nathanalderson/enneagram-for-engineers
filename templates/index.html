Closures

A closure is an object consisting of a function and the environment (local variables) in which the function was created. Basically, it's function that remembers its context.

function makeIncrementor(i) {
  return function(x) {
    return x+i;
  }
}

f1 = makeIncrementor(1);
f2 = makeIncrementor(2);
f1(1);	// returns 2
f2(1);	// returns 3
